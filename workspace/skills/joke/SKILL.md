# Joke Skill

Tells random jokes when asked. Lightens the mood!

## Usage
- `tell me a joke` - Get a random joke
- `make me laugh` - Get a funny joke
- `cheer me up` - Get a lighthearted joke

## How it works
Uses a collection of jokes from various categories:
1. Programming jokes
2. Dad jokes
3. Puns
4. Knock-knock jokes

## Examples
```python
# When user says "tell me a joke"
return "Why do programmers prefer dark mode? Because light attracts bugs!"
```

## Dependencies
None - just humor!

## Notes
- Jokes are family-friendly
- Can be customized with user preferences
- Random selection ensures variety
